## Verdict

**PASS for local gate candidate; LB gate remains mandatory.** Best variant is an AU-only `char_wb 3-5` LinearSVC with `C=1.0`, trained only on nonholdout AU rows, and applied to AU holdout rows as a soft route:

`P_final = 0.9 * P_au + 0.1 * P_blend`

This is still a local league result on an AU-routing axis, structurally close to the prior encoder-share mirage family. Treat it as an LB-gate candidate, not a submit/staging change. No tracked or non-ignored `submit/` source/model files were modified; reviewer noted only an ignored `submit/__pycache__` generated by importing `au_route.py`.
